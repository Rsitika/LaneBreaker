Concepts learned today -
1. What is Raylib ?
Raylib is an open-source C/C++ game development library used for graphics rendering, input handling, audio, and game development.

# Why did I choose Raylib?
Beginner friendly
Lightweight
Easy to learn
Perfect for 2D games
Works well with C++

2. What is GCC?
GCC (GNU Compiler Collection) is a compiler that converts C++ source code into executable machine code.
example-
main.cpp
   ↓
g++
   ↓
LaneBreaker.exe

3. What is MSYS2?
MSYS2 is a development environment and package manager for Windows that helps install and manage tools such as GCC and Raylib.

# Why did we install it?
To get a modern compiler
To install Raylib easily
To manage libraries

4. What is UCRT64?
UCRT64 is a modern MSYS2 environment that provides a GCC compiler and development tools compatible with modern Windows systems.

# Why are we using it?
Better compatibility
Modern compiler support
Easy library management

5. What is a Game Loop?
A game loop continuously updates and redraws the game until the player closes the game window.
example-
while (!WindowShouldClose())
{
    Update();
    Draw();
}

# Why is it important?
Every game uses a game loop.

Examples:
GTA
Minecraft
Temple Run
Subway Surfers
